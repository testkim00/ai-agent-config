# File Processor Template

파일 단위 처리, 변환, 생성 같은 제한된 작업을 worker에게 맡길 때 쓰는 brief 템플릿이다.

## 권장 brief

```text
담당 파일:
- {input/output paths}
처리 목표: {변환/정리/생성}
입력 조건:
- 파일 존재 여부
- 형식 또는 인코딩 조건
작업 규칙:
- 지정된 파일만 다룰 것
- 예상치 못한 포맷이면 중단 후 보고할 것
검증:
- 출력 파일 생성 확인
- 핵심 내용 또는 형식 확인
최종 보고:
- 처리 파일
- 처리 결과
- 검증 결과
```

## 적합한 작업

- 포맷 변환
- 파일 세트 정리
- 제한된 경로 내의 기계적 수정

## 부적합한 작업

- 전역 설계가 필요한 리팩터링
- 어떤 파일이 정답인지 먼저 분석이 필요한 경우
