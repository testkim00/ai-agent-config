# MD Cleanup

마크다운 파일의 중복을 제거하고 내용을 정리합니다.

## 사용법

```
/md-cleanup {파일경로}
/md-cleanup {파일경로} --dry-run  # 미리보기만
```

## 처리 흐름

### 1. 파일 분석
- 대상 파일 읽기
- 구조 파악 (섹션, 목록, 테이블 등)

### 2. 중복 제거
- 동일하거나 유사한 내용 병합
- 반복되는 설명 통합
- 중복 항목 제거

### 3. 구조 정리
- 논리적 순서로 재배치
- 관련 내용 그룹화
- 일관된 포맷 적용

### 4. 정리 기준
- 제목 계층 정리 (h1 → h2 → h3 순서)
- 빈 줄 정규화 (연속 빈 줄 → 단일 빈 줄)
- 목록 스타일 통일
- 테이블 정렬
- 불필요한 공백 제거

### 5. 결과 출력
- 변경 전/후 비교 요약
- 제거된 중복 항목 목록
- 정리된 파일 저장

## 주의사항

- 원본 의미가 손실되지 않도록 주의
- 의도적인 반복(예: 강조)은 유지
- 불확실한 경우 사용자에게 확인

## 예시

```bash
/md-cleanup ~/.claude/skills/subagent-convention/instruction.md
```

**출력:**
```
[분석] instruction.md (150줄)
[중복] 3개 항목 발견
  - "병렬 실행 조건" 2회 반복 → 통합
  - "핵심 원칙" 유사 내용 → 병합
[정리] 완료 (150줄 → 120줄, 20% 감소)
[저장] instruction.md 업데이트됨
```
