# Tidy 커맨드 인덱스

> **수정 시 필수 확인**
>
> Tidy 커맨드 수정 전:
> 1. 이 파일의 관련 문서 목록 확인
> 2. `_common.md` 공통 규칙 확인
> 3. 개별 커맨드와 공통 규칙 일관성 유지

## 문서 구조

```
commands/tidy/
├── _index.md       ← 현재 파일
├── _common.md      ← 공통 규칙 (모든 tidy 커맨드 적용)
└── md.md           ← 마크다운 정리
```

## 커맨드 목록

| 커맨드 | 파일 | 설명 |
|--------|------|------|
| `/tidy:md` | `md.md` | 마크다운 파일 정리 |
| (공통) | `_common.md` | 모든 tidy 커맨드에 적용되는 공통 규칙 |

## 공통 규칙 (`_common.md`)

- 파일 형식별 정리 규칙
- 들여쓰기, 줄바꿈 규칙
- 특수문자 처리

## 수정 시 체크리스트

- [ ] 공통 규칙 변경 → `_common.md` 수정 후 개별 커맨드 확인
- [ ] 새 형식 추가 → `{format}.md` 생성 + 이 파일 업데이트
- [ ] 마크다운 규칙 변경 → `md.md` 수정

## 확장 가이드

새로운 파일 형식 정리 커맨드 추가 시:

```
1. commands/tidy/{format}.md 생성
2. _common.md 규칙 상속 여부 결정
3. 이 파일(_index.md)에 등록
4. commands/_index.md에도 등록
```
